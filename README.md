# PonSarYay - ပုံစာရေး - Myanmar Text-on-Image Editor

**ပုံစာရေး (PonSarYay)** သည် ဓာတ်ပုံများပေါ်တွင် မြန်မာစာနှင့် အင်္ဂလိပ်စာများကို အဆင့်မြင့်ပြီး လှပစွာ ရေးသားနိုင်ရန် ဖန်တီးထားသည့် ဝဘ်အက်ပလီကေးရှင်း (Web App) တစ်ခု ဖြစ်ပါသည်။ ဆိုရှယ်မီဒီယာ ပိုစ့်များ၊ ကောက်နုတ်ချက်များနှင့် ကြော်ငြာများကို မြန်မာဖောင့်အမျိုးမျိုး အသုံးပြု၍ စိတ်ကြိုက် ပြင်ဆင်နိုင်စေရန် ရိုးရှင်းပြီး ထိရောက်သောအသွင်အပြင်ဖြင့် တည်ဆောက်ဖန်တီးထားပါသည်။ 
ကွန်ပျူတာဖြင့်သာမက ဖုန်းဖြင့်ပါ အလွယ်တကူ အသုံးပြုနိုင်စေရန် စနစ်တကျ ပြုလုပ်ထားပါသည်။

**PonSarYay** is a modern, high-performance web application designed for adding beautiful Myanmar and English text to your images. With a wide collection of handcrafted Myanmar fonts and robust text styling tools, creators can easily design high-quality social media content, quotes, and professional marketing graphics. It is built to be fast, responsive, and incredibly user-friendly on both desktop and mobile devices.

---

## Key Features (အဓိက ပါဝင်သည့် လုပ်ဆောင်ချက်များ)

- **Multi-Font Support**: Hand-picked Myanmar (Unicode) and English font collections. (မြန်မာ ယူနီကုဒ်ဖောင့်များနှင့် အင်္ဂလိပ်ဖောင့်များစွာ ပါဝင်ခြင်း)
- **Precision Text Editing**: Adjust size, color, alignment, line height, and opacity with ease. (စာသားအရွယ်အစား၊ အရောင်၊ နေရာချထားမှု၊ စာကြောင်းအကွာအဝေးနှင့် အကြည်အလင်းတို့ကို လွယ်ကူစွာ ချိန်ညှိနိုင်ခြင်း)
- **Image Tools**: Built-in image upload, cropping, and canvas clearing functions. (ဓာတ်ပုံများ တင်ခြင်း၊ ဖြတ်တောက်ခြင်းနှင့် ဖျက်ခြင်းများ ပြုလုပ်နိုင်ခြင်း)
- **Undo/Redo**: Full history support to quickly fix any editing mistakes. (အမှားများရှိပါက အလွယ်တကူ နောက်ပြန်ဆုတ်နိုင်ခြင်း)
- **Smart Export**: Download your creations as high-quality PNG or JPEG, auto-cropped to your image area. (ပုံများကို အရည်အသွေးမြင့် PNG သို့မဟုတ် JPEG အဖြစ် ဒေါင်းလုဒ်ရယူနိုင်ခြင်း)
- **Responsive Layout**: Seamless experience across mobile, tablet, and desktop views. (ဖုန်း၊ တက်ဘလက်နှင့် ကွန်ပျူတာ အားလုံးတွင် အဆင်ပြေစွာ အသုံးပြုနိုင်ခြင်း)
- **Dynamic Themes**: Beautiful Light and Dark modes to suit your preference. (အလင်းနှင့် အမှောင် Theme နှစ်မျိုးလုံး အသုံးပြုနိုင်ခြင်း)
